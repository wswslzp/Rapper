# Rapper Phase 1: MVP 实现计划

**Goal:** 实现 Rapper 的基础框架，能够启动 Claude Code 并通过 bash-runner MCP 安全执行命令
**Architecture:** Bash 入口脚本 + FastMCP Python 服务器 + uv 环境管理
**Tech Stack:** Bash, Python 3.12+, FastMCP, uv

---

## Task 1: 项目初始化

### 1.1 创建 pyproject.toml 和 uv 环境

**File:** `/app/rapper/pyproject.toml`

```toml
[project]
name = "rapper"
version = "0.1.0"
description = "Claude Code wrapper with safety guardrails"
requires-python = ">=3.11"
dependencies = [
    "mcp>=1.0.0",
    "pyyaml>=6.0",
]

[project.optional-dependencies]
dev = [
    "pytest>=8.0",
]
```

**Commands:**
```bash
cd /app/rapper
uv sync
```

**Verification:** `uv run python --version` 返回 Python 版本

## Task 2: bash-runner MCP 实现

### 2.1 创建 MCP 服务器骨架

**File:** `/app/rapper/mcp-servers/bash-runner/server.py`

从 Pixel 移植，简化版本：
- `run_bash(command, timeout, workdir, background, auto_background)` 工具
- 危险命令阻止 (BLOCKED_PATTERNS)
- 异步执行避免阻塞

**Verification:**
```bash
cd /app/rapper
echo '{"jsonrpc":"2.0","method":"initialize","params":{"protocolVersion":"2024-11-05","capabilities":{},"clientInfo":{"name":"test"}},"id":1}' | uv run python mcp-servers/bash-runner/server.py
```

---

### 2.2 实现危险命令检测

从 Pixel 的 `bash_mcp.py` 移植 `BLOCKED_PATTERNS`：
- rm -rf, sudo, su root
- fork bomb, shutdown, reboot
- chmod 777 /, chown /
- netcat listen, nmap

---

### 2.3 实现后台任务支持

- `background=True`: 立即返回 PID
- `auto_background=N`: 运行 N 秒后自动转后台

## Task 3: 入口脚本

### 3.1 创建 rapper 入口脚本

**File:** `/app/rapper/rapper`

功能：
1. 首次运行检测并执行 setup
2. 解析命令行参数
3. 启动 Claude Code (透传参数)

**Commands 支持:**
```bash
rapper                    # 启动交互式
rapper -p "prompt"        # 短任务
rapper --acp --stdio      # ACP 模式
rapper --setup            # 强制重新 setup
rapper --check            # 检查状态
```

**Verification:**
```bash
chmod +x /app/rapper/rapper
/app/rapper/rapper --check
```

---

### 3.2 实现 setup 流程

Setup 步骤：
1. 创建 `~/.rapper/` 目录
2. 生成默认 `config.yaml`
3. 运行 `uv sync` 安装依赖
4. 注册 MCP 到 `~/.claude.json`
5. 配置 `~/.claude/settings.json` (如需要)

---

### 3.3 创建 PATH 软链接脚本

**File:** `/app/rapper/bin/rapper`

```bash
#!/bin/bash
exec /app/rapper/rapper "$@"
```

用户可以将 `/app/rapper/bin` 加入 PATH。

## Task 5: Claude Code 集成

### 5.1 MCP 注册

修改 `~/.claude.json` 添加 bash-runner：

```json
{
  "mcpServers": {
    "bash-runner": {
      "command": "uv",
      "args": ["--directory", "/app/rapper", "run", "python", "mcp-servers/bash-runner/server.py"]
    }
  }
}
```
