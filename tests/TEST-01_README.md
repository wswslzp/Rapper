## Implementation Requirements (IMPL-01)

### Core Changes Needed
1. **Replace hardcoded polling** with configurable `poll_columns`
2. **Add config support** for `agent_board.poll_columns` 
3. **Maintain backward compatibility** - default to `['todo', 'ready']`
4. **Support edge cases** - empty list, single column, multiple columns

### Expected Config Examples

**Rapper Config:**
```yaml
agent_board:
  role: rapper
  poll_columns: ['todo', 'ready']  # Default behavior
```

**Reviewer Config:**
```yaml
agent_board:
  role: reviewer
  poll_columns: ['review']  # Only review column
```

**Legacy Config (no poll_columns):**
```yaml
agent_board:
  # Should default to ['todo', 'ready'] for backward compatibility
```

def _poll_and_execute_tasks(self):
    # Get poll_columns from config, default to ['todo', 'ready']
    poll_columns = self.config.get('agent_board', {}).get('poll_columns', ['todo', 'ready'])
    
    # Handle edge cases
    if not poll_columns or not isinstance(poll_columns, list):
        poll_columns = ['todo', 'ready']  # Fallback
    
    # Poll configured columns
    all_tasks = []
    for column in poll_columns:
        tasks = self.client.get_tasks(None, column)
        all_tasks.extend(tasks)
