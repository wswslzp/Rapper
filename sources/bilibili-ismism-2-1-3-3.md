## Summary

伊壁鸠鲁发展了原子论，通过原子偏转(swerve)理论调和必然性与自由意志的对立，建立智性享乐主义体系。他区分三种欲望类型，追求静态快乐而非动态快乐，认为真正的快乐是不害怕、不痛苦的宁静状态。伊壁鸠鲁主张通过哲学思考与不干涉的神发生关联，获得最高层次的智性快乐，同时通过"我存在时未死，死时已不存在"的逻辑消解对死亡的恐惧。

## Key Points

- **原子偏转理论**: 原子在下落过程中会突然偏转，打破了德谟克利特严格的决定论
- **三分法哲学**: 世界现象分为必然的、偶然的、能动的，为伦理和自由留出空间
- **欲望分类**: 必要且自然(饮食居住)、非必要但自然(美食肉欲)、既非必要也非自然(权力地位)
- **静态快乐**: 相对于动态快乐，追求不害怕不痛苦的宁静状态，通过哲学思考获得
- **不干涉的神**: 神是完美抽象的存在，不降神罚也不赐福运，只在哲学活动中与人发生关联
- **死亡哲学**: "我存在时未死，死时已不存在"，死亡不可体验故无需恐惧
- **经验归纳法**: 通过相似性类比进行经验归纳，是经验科学的先驱

## Concepts

- [[智性享乐主义]]: 通过理性和哲学思考获得精神快乐的享乐主义
- [[原子偏转]]: 原子突然改变运动方向，为自由意志提供本体论基础
- [[静态快乐]]: 摆脱痛苦和恐惧的宁静状态，区别于感官的动态快乐
- [[三种欲望]]: 必要自然、非必要自然、既非必要也非自然的欲望分类体系
- [[不干涉的神]]: 完美但不干预现实世界的神，只在智性活动中提供快乐
- [[伊壁鸠鲁悖论]]: 关于死亡不可恐惧的哲学论证
