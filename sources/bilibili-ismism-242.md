# 实证主义（2-4-2）

## Summary

为人性筑起圣坛，科学哲学、社会学创始人孔德的形而上学。实证主义作为一种理论体系，将人类历史分为神学、形而上学和实证三个阶段，强调从人类视角出发把握一切知识，最终建立以人道教为核心的新等级制社会。

## Key Points

### 三阶段论
- **神学阶段**：人将未知投射给全能的外在超越者
- **形而上学阶段**：人将未知投射为内在可感知的理念
- **实证阶段**：通过外在经验事实解释一切，拒斥形而上学

### 认识论架构
- **社会学vs科学**：社会学是关于科学的哲学
- **形式vs内容**：方法与理论的二分支撑科学阶段论
- **人类视角**：所有知识都从human perspective出发

### 社会理论
- **新等级制**：实证主义社会需要政府外部干涉和精神力量
- **人道教**：对人类本身的崇拜，女性和普罗大众具有超越性力量
- **时间观**：现在是过去与未来斗争的休战结果

## Concepts

1. **实证主义** - 强调经验和科学方法，拒斥形而上学的哲学立场
2. **三阶段论** - 人类社会从神学经形而上学到实证的历史发展模式
3. **社会学** - 孔德创立的学科，要求掌握一切知识的全能视角
4. **科学哲学** - 从人性出发把握一切学科的神仙般学问
5. **人道教** - 对人类本身的宗教性崇拜
6. **孔德** - 实证主义和社会学的创始人，真性情但理论粗糙的思想家
