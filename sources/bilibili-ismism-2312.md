# 运动的静止主义（2-3-1-2）

## Summary

布鲁诺究竟因何而死？——运动的静止主义对亚里士多德体系重拳出击，启发了牛顿和伽利略的哲学。布鲁诺提倡的不是日心说，而是宇宙无限说，通过辩证的一元论反对等级制世界观。

## Key Points

### 辩证的一元论
- **对立原则**：认为对立面可以相互转化
- **魔法转化**：将转化过程视为神秘的魔法和启示
- **极大极小**：通过思考极大与极小的一瞬间转化获得神秘启示

### 宇宙无限说
- **非日心说**：布鲁诺坚持的是宇宙无限说，而非日心说
- **无数星系**：宇宙中有无数个星系和世界
- **反等级制**：反对圈层世界的等级制结构

### 反亚里士多德主义
- **反圈层说**：反对亚里士多德的圈层世界观
- **反等级制**：与奥卡姆、库萨的尼古拉一样反对等级制
- **静态世界观**：通过对立转化认为整个世界是静态的

## Concepts

1. **运动的静止主义** - 通过对立转化达到的静态世界观
2. **辩证的一元论** - 布鲁诺的哲学体系，强调对立面的相互转化
3. **布鲁诺** - 意大利哲学家，因宇宙无限说被宗教法庭烧死
4. **宇宙无限说** - 认为宇宙无限，有无数个世界和星系
5. **魔法** - 布鲁诺对对立转化过程的神秘主义理解
6. **对立转化** - 极大与极小等对立面的相互转换
