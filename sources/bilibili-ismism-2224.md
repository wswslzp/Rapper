# 实用的神秘主义（2-2-2-4）

## Summary

韩非子的道家魔法，战国顶级贵族子弟接受的神秘学教育。韩非子通过解读老子的"道德经"，建构了一套实用的神秘主义哲学体系，将"道"理解为一种时间性的法力值，是万物存在的燃料和转化力量。

## Key Points

### 核心概念
- **道者万物之所然也**：道是万物的燃料，万物通过燃烧道来获得存在
- **万物之所积也**：道可以迟滞万物，将其停留在某种状态
- **道理之考也**：道是理的死掉老爸，理是已经生成的空间性结构

### 哲学架构
- **道**：时间性力量，最低限度空间化的时间，给出存在的法力
- **理**：空间性规定，事物的形状、性状和结构
- **时间vs空间**：时间赋予存在力量，空间赋予结构形式

### 神秘主义实践
- **前识**：预言能力，通过修道获得的神秘技能
- **修行方法**：韩非传承的具体修炼功法
- **实用导向**：批判华而不实的神秘主义，追求富国强兵的实用价值

## Concepts

1. **实用的神秘主义** - 韩非子将老子哲学转化为实用的政治和修行工具
2. **道** - 时间性的本体力量，万物存在的燃料和转化动力
3. **理** - 空间性的规定结构，事物的形状和性状
4. **韩非子解老** - 对老子道德经的法家式解读和改造
5. **时间性力量** - 区别于现代物理时间的生成性、事件化时间概念
6. **法术** - 道作为法力值的运用，类似游戏中的魔法系统
