# 救世主义（1-2-4-2）

## Summary

尼采因何而疯？最危险、最廉价的意识形态，意识产生之前的人格化的符号学机制：救世主义。救世主义是一种跨越各大宗教传统的意识形态，通过设立拯救者角色来建构善恶循环的因果网络，濒疯的尼采正是这种机制的体现。

## Key Points

### 尼采与救世主义
- **看哪斯人**：尼采晚年自称克里斯特，体现拯救者面对不值得拯救者的痛苦
- **历史错位**：尼采所处时代不需要布道者，而需要演讲煽动家
- **符号学机制**：尼采本人就是这种机制，而非被其俘获

### 跨宗教现象
- **犹太教**：受膏油的王（Messiah），大卫世系的拯救者
- **琐罗亚斯德教**：Saoshyant，恢复秩序者
- **佛教**：弥勒佛，未来佛
- **道教**：木子弓口（李弘），太上老君下凡

### 结构特征
- **场域论**：一，完全的因果网络善恶循环
- **拯救逻辑**：设立超越性拯救者角色
- **人格化机制**：将抽象力量人格化为具体救世主

## Concepts

1. **救世主义** - 各大宗教传统中的拯救者意识形态
2. **Messianism** - 以设立拯救者为核心的符号学机制
3. **尼采** - 濒疯的哲学家，救世主义机制的体现者
4. **看哪斯人** - 尼采晚年作品，拯救者的痛苦自述
5. **符号学机制** - 意识产生之前的人格化机制
6. **拯救者** - 被设立为超越性角色的救世主形象
