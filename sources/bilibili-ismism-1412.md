# 客观主义（1-4-1-2）

## Summary

最赤裸裸的精英主义，从母国叛逃，连康德都弄不明白的骗子哲学家安·兰德，无良出版商的摇钱树。客观主义是一种前反思的自然主义立场，通过拒斥本体论反思，将庸人的无脑状态包装成精英的高贵形态。

## Key Points

### 基本架构
- **本体论**：四（单纯存在，拒斥本体论反思）
- **认识论**：二（自然现象，前反思状态）
- **目的论**：二（精英vs大众，高贵vs低贱的对立）

### 三大公理
1. **Existence exists**：外在的东西确实外在存在
2. **Consciousness exists**：意识也确实存在（讨好科学共同体）
3. **Existence is identity**：存在即认同，本体论闭合

### 本质特征
- **反智主义**：取消哲学和存在主义反思
- **存在论闭合**：将本体性和本体论等同，自我取消存在论
- **精英主义包装**：将普通人的前反思状态夸成精英的高贵

## Concepts

1. **客观主义** - 安·兰德的哲学体系，实质是前反思的自然主义
2. **精英主义** - 无任何理论基础的赤裸裸优越感表达
3. **安·兰德** - 从苏联叛逃到美国的骗子哲学家
4. **前反思的自然主义** - 拒绝深入思考的庸人立场
5. **反智主义** - 反对哲学思辨和本体论反思的立场
6. **存在论闭合** - 将存在者和存在论混同，取消哲学反思空间
